# Prime Health 💚

Prime Health is a simple and user-friendly calorie tracking and weight loss application designed to help people build healthier eating habits.

## Features
- 🍽️ Food calorie tracking
- 📊 Weight progress tracking
- 💧 Water intake tracking
- 🏃 Activity logging
- 🥗 Large food database with Indian foods
- 📈 Nutrition insights

Prime Health is currently in active development, with new foods and features being added regularly based on tester feedback.
